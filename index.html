WatchGuard Toolbox Project¶

This project aims to collect useful scripts for managing WatchGuard Fireboxes.

Tools¶

  • xmlcheck - a tool for listing/finding contents of WatchGuard XML Files
  • certificate-push - a tool for pushing certificates to WatchGuard Fireboxes; useful especially in combination with letsencrypt
