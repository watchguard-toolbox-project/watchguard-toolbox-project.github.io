WatchGuard Toolbox Project¶

This project aims to collect useful scripts for managing WatchGuard Fireboxes.

Tools¶

  • xmlcheck - listing/finding contents of WatchGuard XML Files

  • watchguard-letsencrypt-deploy - certificate-push - a tool for pushing certificates to WatchGuard Fireboxes; useful especially in combination with letsencrypt

  • fail2ban-for-watchguard - use fail2ban on linux to manipulate blocked-site-list
